Nesting Button Groups & Dropdown Menus

Combining and nesting 2 concepts. Button Groups and Dropdown Menus.
Following classes and attributes are used. (Some of which will be coded later)

  • .dropdown-toggle
    • data-bs-toggle attribute
    • dropdown --- Value for the above "data-bs-toggle" attribute.
  • .dropdown-menu
    • .dropdown-item

The dropdown button group will be nested separately inside the main button group.
