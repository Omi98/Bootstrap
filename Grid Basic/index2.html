Responsive Columns

From what I have understood...

  • There are 12 column units by default on a screen, regardless of its size.
  • There are 6 default grid tiers in Bootstrap v5.0. And those are.
    1. xs - <576px - extra small
    2. sm - ≥576px - small
    3. md - ≥786px - medium
    4. lg - ≥992px - large
    5. xl - ≥1200px - extra large
    6. xxl - ≥1400px - extra extra large

For example, we will take a look at col-3 and col-sm-3 classes.

  • col-3 class is used to create a column that always takes up 3 units of the available width on all screen sizes.
  • col-sm-3 class is used to create a column that takes up 3 units of the available width only on small screens and large. (For screens sm and upto xxl). For screen sizes smaller than small (like xs), it will take up the entire width of screen.
