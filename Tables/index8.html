Contextual Classes

Contextual classes can be used to color the entire table <table>, table rows <tr>, or table cells <td>.


What are Contextual Classes?

Bootstrap5 classes that can be used to provide "meaning through colors".


Following classes can be used:

  • .table-primary
  • .table-success
  • .table-danger
  • .table-info
  • .table-warning
  • .table-active
  • .table-secondary
  • .table-light
  • .table-dark
