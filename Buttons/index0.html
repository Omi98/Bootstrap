Button Styles

Following are the simple styles of Bootstrap5 buttons.

  • Always add the main class .btn to the button element.
  • Add more classes for the contextual button styles.

Contextual Buttons

Following are the classes for the contextual buttons.

  • .btn-primary
  • .btn-secondary
  • .btn-success
  • .btn-info
  • .btn-warning
  • .btn-danger
  • .btn-dark
  • .btn-light
  • .btn-link A button that is a also a link.
Where else can we use btn classes?
The button classes can be used with <a>, <button> or <input> elements.
