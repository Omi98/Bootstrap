Button Outline

Bootstrap also provides 8 outlined/bordered buttons.
Move the mouse over them to see an additional hover

Following are the classes for the outline buttons. Use them in addition to the basic .btn class.

  • .btn-outline-primary
  • .btn-outline-secondary
  • .btn-outline-success
  • .btn-outline-info
  • .btn-outline-warning
  • .btn-outline-danger
  • .btn-outline-dark
  • .btn-outline-light
Note:
For the .btn-outline-light class for Light button, make sure to add .text-dark for the text color change. Otherwise the text of button will be light (invisible).
