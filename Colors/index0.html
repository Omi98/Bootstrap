Contextual Colors (Meaning Through Colors)

  1. .text-muted class:
    This text is muted.
  2. .text-primary class:
    This text is important.
  3. .text-success class:
    This text indicates success.
  4. .text-info class:
    This text represents some information.
  5. .text-warning class:
    This text represents a warning.
  6. .text-danger class:
    This text represents danger.
  7. .text-secondary class:
    This is secondary text.
  8. .text-dark class:
    This text is dark grey.
  9. .text-body class:
    Default body color (often black).
  10. .text-light class:
    This text is light grey (on black background).
  11. .text-white class:
    This text is white (on black background).
