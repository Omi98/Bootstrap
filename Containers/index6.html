Normal Container

This container has top-margin of 3rem (mt-5) and top-padding of 1.5rem (pt-3).
This is the default .container class. It will have 100% width until the window is bigger than small screen size (576px).
Below are the responsive containers. Resize the browser window to see the effect.

.container-sm

Behaves in the same way as the default container. It has 100% width until the window becomes bigger than extra small screen size (less than 576px).

.container-md

This container has 100% until the window becomes bigger than the small screen size (576px).

.container-lg

This container has 100% until the window becomes bigger than the medium screen size (768px).

.container-xl

This container has 100% screen size till the window becomes bigger than large screen size (992px).

.container-xxl

This container has 100% screen size till the window becomes bigger than the extra large (xl) screen size (1140px).

Fluid Container

Always takes 100% of screen size. Automatically wraps around the text.
