Pagination Alignment

Use utility classes to change the alignment of the pagination blocks.
Add the utility class to the main "ul" element.

  • left-aligned by default. Using .pagination class only.
  • .justify-content-center for center-aligned pagination block.
  • .justify-content-end for right-aligned pagination block.
