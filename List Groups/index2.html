List Groups with Linked Items

To create a list group with linked items:

  • Use div as container element instead of "ul" element. Use .list-group class.
  • Use a tag as child element instead of "li" element. Use .list-group-item class.
  • Optionally, Use .list-group-item-action class inside "a" tag for grey background color on hover.
